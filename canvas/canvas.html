🎨 Canvas Tkinter - Dessiner avec la souris

Créer une application de peinture

Introduction au Canvas

Le Canvas (toile en français) est un widget Tkinter qui permet de dessiner : lignes, rectangles, cercles, textes, et bien plus encore. C'est comme une feuille blanche sur laquelle tu peux créer tes propres dessins !

Avec le Canvas, tu peux créer :

  • Des applications de dessin (comme Paint)
  • Des jeux graphiques
  • Des graphiques et diagrammes
  • Des animations

Créer un Canvas

Code minimal

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Mon Canvas")

# Créer un Canvas
canvas = tk.Canvas(fenetre, width=600, height=400, bg="white")
canvas.pack()

fenetre.mainloop()

Explication :

  • tk.Canvas(fenetre, ...) : Crée le Canvas dans la fenêtre
  • width=600, height=400 : Définit la taille (largeur × hauteur en pixels)
  • bg="white" : Définit la couleur de fond
  • canvas.pack() : Affiche le Canvas

Dessiner des formes sur le Canvas

Ligne

canvas.create_line(x1, y1, x2, y2, fill="black", width=2)

Dessine une ligne du point (x1, y1) au point (x2, y2).

Rectangle

canvas.create_rectangle(x1, y1, x2, y2, fill="blue", outline="black")

Dessine un rectangle entre deux coins opposés.

Ovale/Cercle

canvas.create_oval(x1, y1, x2, y2, fill="red", outline="black")

Dessine un cercle ou une ellipse.

Exemple : Dessiner quelques formes

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Formes sur Canvas")

canvas = tk.Canvas(fenetre, width=400, height=300, bg="white")
canvas.pack()

# Dessiner une ligne
canvas.create_line(50, 50, 200, 50, fill="black", width=3)

# Dessiner un rectangle
canvas.create_rectangle(50, 100, 150, 200, fill="blue", outline="black")

# Dessiner un cercle
canvas.create_oval(200, 100, 300, 200, fill="red", outline="black")

fenetre.mainloop()

Couleurs et styles

Couleurs disponibles

Tu peux utiliser des noms de couleurs en anglais :

"red", "blue", "green", "yellow", "orange", "purple", "pink", 
"brown", "black", "white", "gray", "lightblue", etc.

Ou des codes hexadécimaux :

"#FF0000"  # Rouge
"#00FF00"  # Vert
"#0000FF"  # Bleu
"#FFA500"  # Orange

Options importantes

  • fill : Couleur de remplissage
  • outline : Couleur du contour
  • width : Épaisseur du trait

Événements de souris

Pour créer une application de dessin, il faut détecter les mouvements de la souris et dessiner à chaque position.

Les événements principaux

Événement Description Quand se déclenche-t-il ?
<Button-1> Clic gauche Quand on clique avec le bouton gauche
<B1-Motion> Glisser (clic maintenu) Quand on déplace la souris en maintenant le clic gauche
<ButtonRelease-1> Relâcher le clic Quand on relâche le bouton gauche
<Motion> Mouvement de souris Quand la souris bouge (même sans clic)

La méthode bind()

La méthode bind() permet de lier un événement (comme un clic) à une fonction qui sera exécutée quand cet événement se produit.

Syntaxe

canvas.bind("<événement>", fonction)

Récupérer la position de la souris

Quand un événement souris se déclenche, Python passe automatiquement un objet event qui contient la position de la souris :

  • event.x : Position horizontale (x)
  • event.y : Position verticale (y)

Exemple : Afficher la position de la souris

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Position de la souris")

canvas = tk.Canvas(fenetre, width=400, height=300, bg="white")
canvas.pack()

def afficher_position(event):
    print(f"Position : x={event.x}, y={event.y}")

# Lier l'événement de mouvement à la fonction
canvas.bind("<Motion>", afficher_position)

fenetre.mainloop()

Dessiner en bougeant la souris

Principe

Pour créer un dessin libre (comme avec un pinceau), on dessine des petits cercles à chaque position de la souris quand le bouton est maintenu enfoncé.

Exemple simple : Dessiner au clic maintenu

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Dessin simple")

canvas = tk.Canvas(fenetre, width=600, height=400, bg="white")
canvas.pack()

def dessiner(event):
    # Dessiner un petit cercle à la position de la souris
    x, y = event.x, event.y
    rayon = 3
    canvas.create_oval(x-rayon, y-rayon, x+rayon, y+rayon, 
                       fill="black", outline="black")

# Dessiner seulement quand on glisse avec le clic maintenu
canvas.bind("<B1-Motion>", dessiner)

fenetre.mainloop()
💡 <B1-Motion> signifie "Button 1 Motion" : la souris bouge pendant que le bouton gauche est maintenu enfoncé.

Amélioration : Dessiner des lignes continues

Pour un dessin plus fluide, on peut dessiner des lignes entre les positions successives au lieu de petits cercles. 

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Dessin avec lignes")

canvas = tk.Canvas(fenetre, width=600, height=400, bg="white")
canvas.pack()

# Variables pour mémoriser la dernière position
derniere_x = None
derniere_y = None

def commencer_dessin(event):
    global derniere_x, derniere_y
    derniere_x = event.x
    derniere_y = event.y

def dessiner(event):
    global derniere_x, derniere_y
    if derniere_x and derniere_y:
        # Dessiner une ligne de la dernière position à la position actuelle
        canvas.create_line(derniere_x, derniere_y, event.x, event.y,
                          fill="black", width=3, capstyle="round", smooth=True)
    # Mettre à jour la dernière position
    derniere_x = event.x
    derniere_y = event.y

def arreter_dessin(event):
    global derniere_x, derniere_y
    derniere_x = None
    derniere_y = None

# Lier les événements
canvas.bind("<Button-1>", commencer_dessin)
canvas.bind("<B1-Motion>", dessiner)
canvas.bind("<ButtonRelease-1>", arreter_dessin)

fenetre.mainloop()
⚠️ Points clés :
  • <Button-1> : Déclenché quand on commence à cliquer
  • <B1-Motion> : Déclenché pendant qu'on glisse avec le clic maintenu
  • <ButtonRelease-1> : Déclenché quand on relâche le clic
  • On utilise des variables globales pour mémoriser la dernière position

Exemple complet : Application de peinture basique

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Application de peinture")

# Créer le Canvas
canvas = tk.Canvas(fenetre, width=800, height=600, bg="white")
canvas.pack()

# Variables globales
derniere_x = None
derniere_y = None
couleur_actuelle = "black"

def commencer_dessin(event):
    global derniere_x, derniere_y
    derniere_x = event.x
    derniere_y = event.y

def dessiner(event):
    global derniere_x, derniere_y
    if derniere_x and derniere_y:
        canvas.create_line(derniere_x, derniere_y, event.x, event.y,
                          fill=couleur_actuelle, width=5, 
                          capstyle="round", smooth=True)
    derniere_x = event.x
    derniere_y = event.y

def arreter_dessin(event):
    global derniere_x, derniere_y
    derniere_x = None
    derniere_y = None

def changer_couleur_noir():
    global couleur_actuelle
    couleur_actuelle = "black"

def changer_couleur_rouge():
    global couleur_actuelle
    couleur_actuelle = "red"

def changer_couleur_bleu():
    global couleur_actuelle
    couleur_actuelle = "blue"

def effacer_tout():
    canvas.delete("all")

# Lier les événements de dessin
canvas.bind("<Button-1>", commencer_dessin)
canvas.bind("<B1-Motion>", dessiner)
canvas.bind("<ButtonRelease-1>", arreter_dessin)

# Créer des boutons pour changer de couleur
bouton_noir = tk.Button(fenetre, text="Noir", command=changer_couleur_noir, bg="black", fg="white")
bouton_noir.pack(side="left", padx=5)

bouton_rouge = tk.Button(fenetre, text="Rouge", command=changer_couleur_rouge, bg="red", fg="white")
bouton_rouge.pack(side="left", padx=5)

bouton_bleu = tk.Button(fenetre, text="Bleu", command=changer_couleur_bleu, bg="blue", fg="white")
bouton_bleu.pack(side="left", padx=5)

bouton_effacer = tk.Button(fenetre, text="Effacer tout", command=effacer_tout)
bouton_effacer.pack(side="left", padx=5)

fenetre.mainloop()
💡 Astuces :
  • canvas.delete("all") efface tout le contenu du Canvas
  • capstyle="round" rend les extrémités des lignes arrondies
  • smooth=True lisse les lignes pour un rendu plus naturel
  • side="left" dans pack() place les boutons côte à côte

✏️ Exercices pratiques

Exercice 1 - Canvas de base

Consigne :

Crée un programme avec :

  • Une fenêtre de titre "Mon premier Canvas"
  • Un Canvas de taille 500x400 pixels avec un fond blanc
  • Dessine 3 formes de ton choix sur le Canvas (rectangle, cercle, ligne...)
💡 Utilise canvas.create_rectangle(), canvas.create_oval(), canvas.create_line()

Exercice 2 - Dessiner au clic

Consigne :

Crée un programme qui :

  • Affiche un Canvas blanc
  • Dessine un petit cercle noir à chaque endroit où tu cliques avec la souris
💡 Utilise canvas.bind("<Button-1>", fonction) pour détecter les clics.
💡 Dans la fonction, récupère la position avec event.x et event.y, puis dessine un petit cercle avec canvas.create_oval()

Exercice 3 - Application de peinture basique 🎨

Consigne :

Crée une application qui permet de dessiner en faisant glisser la souris :

  • Un Canvas blanc de taille 700x500
  • Quand on glisse la souris avec le clic maintenu, le programme dessine une ligne noire
  • L'épaisseur du trait doit être de 3 pixels
💡 Méthode :
1. Crée deux variables globales derniere_x = None et derniere_y = None
2. Crée 3 fonctions :
   - commencer_dessin(event) : mémorise la position de départ
   - dessiner(event) : dessine une ligne de la dernière position à la position actuelle
   - arreter_dessin(event) : remet les variables à None
3. Lie les événements :
   - <Button-1> → commencer_dessin
   - <B1-Motion> → dessiner
   - <ButtonRelease-1> → arreter_dessin

Exercice 4 - Changer de couleur

Consigne :

Améliore ton application de peinture en ajoutant :

  • 3 boutons en bas de la fenêtre : "Noir", "Rouge", "Bleu"
  • Chaque bouton change la couleur du pinceau
  • Le dessin se fait avec la couleur sélectionnée
💡 Crée une variable globale couleur_actuelle = "black"
💡 Crée 3 fonctions qui changent cette variable (exemple : changer_couleur_rouge())
💡 Dans la fonction dessiner(), utilise fill=couleur_actuelle
💡 Place les boutons avec .pack(side="left") pour les mettre côte à côte

Exercice 5 - Taille du pinceau

Consigne :

Ajoute la possibilité de changer l'épaisseur du trait :

  • Ajoute 3 boutons : "Fin" (2 pixels), "Moyen" (5 pixels), "Épais" (10 pixels)
  • Le trait se dessine avec l'épaisseur sélectionnée
💡 Crée une variable globale epaisseur_actuelle = 3
💡 Utilise width=epaisseur_actuelle dans canvas.create_line()

Exercice 6 - Effacer le dessin

Consigne :

Ajoute un bouton "Effacer tout" qui :

  • Supprime tout ce qui a été dessiné sur le Canvas
  • Remet le Canvas blanc et vide
💡 Utilise canvas.delete("all") pour tout effacer

Projet final - Application de peinture complète 🎨✨

Consigne :

Crée une application de peinture complète qui combine toutes les fonctionnalités :

Fonctionnalités obligatoires :

  • Un Canvas blanc de 800x600 pixels
  • Possibilité de dessiner en faisant glisser la souris
  • Au moins 5 couleurs différentes au choix (noir, rouge, bleu, vert, jaune...)
  • Au moins 3 tailles de pinceau différentes
  • Un bouton "Effacer tout"

Fonctionnalités bonus (optionnelles) :

  • Ajouter plus de couleurs (orange, violet, rose, marron...)
  • Ajouter un bouton "Gomme" qui dessine en blanc
  • Organiser les boutons de manière esthétique (utilise frame pour grouper les boutons)
  • Ajouter un Label qui affiche la couleur et taille actuelles
  • Permettre de dessiner des formes (rectangle, cercle) en cliquant

Conseils pour réussir :

  • Commence par le dessin de base (exercice 3)
  • Ajoute ensuite les couleurs (exercice 4)
  • Puis les tailles de pinceau (exercice 5)
  • Enfin le bouton effacer (exercice 6)
  • Teste ton programme régulièrement !
💡 Structure recommandée :
- Variables globales : derniere_x, derniere_y, couleur_actuelle, epaisseur_actuelle
- Fonctions de dessin : commencer_dessin(), dessiner(), arreter_dessin()
- Fonctions pour les couleurs : une fonction par couleur
- Fonctions pour les tailles : une fonction par taille
- Fonction effacer_tout()
- Canvas en haut, boutons en bas organisés par catégorie
