Les boucles while et for en Python

La boucle while

La boucle while permet de répéter des instructions tant qu’une condition est vraie.

  • On ne sait pas toujours combien de fois la boucle va se répéter
  • La condition est testée à chaque tour
  • La boucle s’arrête lorsque la condition devient fausse
Il est indispensable de faire évoluer la condition, sinon la boucle ne s’arrête jamais.

Exemple

x = 0
while x < 3:
    print("Bonjour")
    x = x + 1

À chaque tour, la valeur de x augmente. Lorsque x atteint 3, la condition devient fausse et la boucle s’arrête.

La boucle for

La boucle for est utilisée lorsque l’on souhaite répéter une action un nombre précis de fois ou parcourir une série d’éléments.

  • La répétition est automatique
  • Aucune condition à modifier manuellement

Exemple simple

for i in range(3):
    print("Bonjour")

Ce programme affiche le mot Bonjour trois fois.

Comprendre la fonction range()

La fonction range() permet de générer une suite de nombres. Elle est très utilisée avec la boucle for pour contrôler le nombre de répétitions.

range(nombre)

range(3)

Cette instruction produit les nombres suivants : 

0, 1, 2
Le dernier nombre n’est jamais inclus. range(3) s’arrête avant 3.

Utilisation avec for

for nombre in range(5):
    print(nombre)

À chaque tour de boucle, la variable reçoit le nombre suivant généré par range().

range(début, fin)

for nombre in range(2, 6):
    print(nombre)

Cette boucle affiche les nombres de 2 à 5.

range(début, fin, pas)

for nombre in range(0, 10, 2):
    print(nombre)

Ici, la boucle avance de deux en deux.

La boucle for sans range()

La boucle for ne sert pas uniquement à compter. Elle peut aussi parcourir directement des éléments.

Parcourir un mot

mot = "Python"

for lettre in mot:
    print(lettre)

Chaque lettre du mot est affichée une par une.

Parcourir une liste

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

Chaque élément de la liste est affiché successivement.

À retenir :
  • while répète tant qu’une condition est vraie
  • for automatise les répétitions
  • range() génère des nombres
  • for peut parcourir nombres, mots ou listes
