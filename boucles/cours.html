Cours – Les boucles while et for en Python

La boucle while

La boucle while permet de répéter des instructions tant qu’une condition est vraie.

  • On ne sait pas toujours à l’avance combien de fois la boucle va se répéter
  • La condition est testée à chaque tour
  • Si la condition devient fausse, la boucle s’arrête
Il est obligatoire de faire changer la condition dans la boucle, sinon elle ne s’arrête jamais.

Exemple

x = 0
while x < 3:
    print("Bonjour")
    x = x + 1

Explication

  • x commence à 0
  • Tant que x < 3 est vrai, le message s’affiche
  • x augmente de 1 à chaque tour
  • Quand x vaut 3, la boucle s’arrête

La boucle for

La boucle for sert à répéter des instructions un nombre précis de fois.

  • On sait à l’avance combien de fois la boucle va tourner
  • Le compteur avance automatiquement
  • On utilise souvent la fonction range()

Qu’est-ce qu’un index ?

Un index est un numéro de position. Il permet de savoir où se trouve un élément. 

fruits = ["pomme", "banane", "orange"]
Index Valeur
0 pomme
1 banane
2 orange
Les index commencent toujours à 0.

Exemple

print(fruits[0])  # affiche pomme
print(fruits[2])  # affiche orange

Qu’est-ce que range() ?

La fonction range() permet de créer une suite de nombres. 

range(3)

Cette instruction correspond aux valeurs :

0, 1, 2
Le dernier nombre n’est jamais inclus.

Utiliser range() avec for

for i in range(3):
    print("Bonjour")

Explication

  • range(3) produit : 0, 1, 2
  • i prend chaque valeur
  • La boucle s’exécute 3 fois

Le rôle de la variable i

for i in range(5):
    print(i)

Résultat :

0
1
2
3
4

La variable i est un index. Elle indique le numéro du tour de boucle.

Lien entre for, range() et les listes

notes = [12, 15, 9]

for i in range(len(notes)):
    print(notes[i])

Ici, i sert d’index pour accéder aux éléments de la liste.

Conclusion

Boucle Utilisation
while Tant qu’une condition est vraie
for Un nombre précis de répétitions

À retenir absolument

  • Un index est un numéro de position
  • Les index commencent à 0
  • range(n) va de 0 à n-1
  • for est utilisé quand on connaît le nombre de répétitions
  • while est utilisé quand on ne le connaît pas
