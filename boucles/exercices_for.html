Exercices – Boucles for et range

Exercice 1 – Répéter un message

Écris un programme qui :

  • affiche le mot Bonjour cinq fois
  • utilise une boucle for

Exercice 2 – Compter de 0 à 4

Écris un programme qui :

  • affiche les nombres de 0 à 4
  • utilise une boucle for

Indice : range(5)

Exercice 3 – Compter de 1 à 5

Écris un programme qui :

  • affiche les nombres de 1 à 5
  • utilise une boucle for

Indice : range(1, 6)

Exercice 4 – Les nombres pairs

Écris un programme qui :

  • affiche les nombres pairs de 0 à 10
  • utilise une boucle for

Indice : avancer de 2 → range(0, 11, 2)

Exercice 5 – Compter à l’envers

Écris un programme qui :

  • affiche les nombres de 5 à 1
  • utilise une boucle for

Indice : range(5, 0, -1)

Exercice 6 – Additionner

Écris un programme qui :

  • calcule la somme des nombres de 1 à 5
  • affiche le résultat

Indice : utiliser une variable somme

Exercice 7 – Avec une variable

Écris un programme qui :

  • crée une variable n
  • affiche les nombres de 0 à n

Exemple : si n = 3 → 0 1 2 3

Exercice 8 – Parcourir un mot

Écris un programme qui :

  • affiche chaque lettre du mot python sur une ligne différente
  • utilise une boucle for

Indice : on peut parcourir un mot directement

Exercice 9 – Défi final (plus difficile)

Écris un programme qui :

  • demande un nombre à l’utilisateur
  • affiche tous les nombres de 0 jusqu’à ce nombre
  • affiche à la fin la somme de ces nombres

Exemple :

Entrer un nombre : 4
0
1
2
3
4
Somme = 10

Cet exercice permet de vérifier que tu sais utiliser :

  • une variable
  • une boucle for
  • la fonction range()
