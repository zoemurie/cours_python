Exercices – Boucle while en Python

Exercice 1 – Bonjour plusieurs fois

Écris un programme bonjour.py qui :

  • crée une variable compteur initialisée à 0
  • affiche le mot Bonjour trois fois
  • utilise une boucle while
  • augmente compteur de 1 à chaque tour

Exercice 2 – Compter de 0 à 5

Écris un programme compter.py qui :

  • crée une variable x = 0
  • affiche tous les nombres de 0 à 5
  • utilise une boucle while

Exercice 3 – Compte à rebours

Écris un programme compte_a_rebours.py qui :

  • crée une variable x = 5
  • affiche : 5, 4, 3, 2, 1
  • diminue x de 1 à chaque tour
  • s’arrête lorsque x vaut 0

Exercice 4 – Boucle infinie corrigée

Voici un programme boucle_infinie.py incomplet :

x = 0
while x > 5:
    print("Salut")
  • explique pourquoi ce programme ne fonctionne pas correctement
  • corrige-le pour qu’il affiche Salut cinq fois

Exercice 5 – Deviner un nombre (simplifié)

Écris un programme deviner_nombre.py qui :

  • choisit un nombre secret : nombre_secret = 7
  • demande à l’utilisateur un nombre proposition
  • tant que la proposition est incorrecte, redemande un nombre
  • affiche Bravo ! quand le nombre est trouvé

Exercice 6 – Petit défi

Écris un programme stop.py qui :

  • demande à l’utilisateur de taper STOP
  • tant que l’utilisateur ne tape pas STOP, affiche Je continue !
  • quand l’utilisateur tape STOP, affiche Fini !
