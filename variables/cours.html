Les variables en Python

Qu’est-ce qu’une variable ?

Une variable est une boîte qui sert à stocker une information. Cette information peut être un nombre, un mot, ou autre chose.

En Python, on utilise les variables pour mémoriser des données et les réutiliser plus tard dans le programme.

Créer une variable

Pour créer une variable, on écrit : 

age = 12

Ici :

  • age est le nom de la variable
  • = sert à donner une valeur
  • 12 est la valeur stockée

Utiliser une variable

On peut afficher le contenu d’une variable avec print : 

age = 12
print(age)

Le programme affiche : 

12

Modifier la valeur d’une variable

Une variable peut changer de valeur pendant le programme : 

score = 10
score = 15
print(score)
La nouvelle valeur remplace l’ancienne.

Les types de variables courants

Les nombres

age = 14
points = 250

Les nombres décimaux

prix = 3.5
moyenne = 12.75

Les textes (chaînes de caractères)

prenom = "Alice"
message = "Bonjour"
⚠️ Les textes doivent toujours être entre guillemets.

Utiliser les variables dans des calculs

a = 5
b = 3
resultat = a + b
print(resultat)

Le programme affiche : 

8

Les variables et input()

On peut stocker une réponse de l’utilisateur dans une variable : 

prenom = input("Quel est ton prénom ? ")
print("Bonjour", prenom)

À retenir absolument

  • Une variable stocke une information
  • On utilise = pour donner une valeur
  • La valeur peut changer
  • Une variable peut contenir un nombre ou un texte
