🖼️ Tkinter

Créer des interfaces graphiques

Introduction à Tkinter

Tkinter est une bibliothèque Python qui permet de créer des interfaces graphiques (fenêtres, boutons, champs de texte, etc.). C'est inclus par défaut avec Python, donc pas besoin d'installer quoi que ce soit !

Avec Tkinter, tu peux créer des programmes avec des fenêtres, comme les applications que tu utilises tous les jours.

Créer une fenêtre

Code minimal

import tkinter as tk

# Créer la fenêtre principale
fenetre = tk.Tk()
fenetre.title("Ma première fenêtre")

# Lancer la boucle principale
fenetre.mainloop()

Explication :

  • import tkinter as tk : Importe la bibliothèque Tkinter
  • fenetre = tk.Tk() : Crée la fenêtre principale
  • fenetre.title("...") : Définit le titre de la fenêtre
  • fenetre.mainloop() : Lance la fenêtre et attend les actions de l'utilisateur
⚠️ Important : mainloop() doit toujours être à la fin ! C'est ce qui garde la fenêtre ouverte.

Personnaliser la fenêtre

# Définir la taille de la fenêtre (largeur x hauteur)
fenetre.geometry("400x300")

# Changer la couleur de fond
fenetre.configure(bg="lightblue")

Les widgets de base

Les widgets sont les éléments que tu ajoutes dans ta fenêtre : boutons, textes, champs de saisie, etc.

Label (afficher du texte)

Un Label permet d'afficher du texte dans la fenêtre.

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Label exemple")

# Créer un label
texte = tk.Label(fenetre, text="Bonjour le monde !")
texte.pack()

fenetre.mainloop()

Personnaliser un Label

texte = tk.Label(
    fenetre,
    text="Mon texte",
    font=("Arial", 20),      # Police et taille
    fg="blue",               # Couleur du texte (foreground)
    bg="yellow"              # Couleur de fond (background)
)
texte.pack()

Button (bouton cliquable)

Un Button est un bouton sur lequel l'utilisateur peut cliquer.

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Bouton exemple")

# Créer un bouton
bouton = tk.Button(fenetre, text="Clique-moi !")
bouton.pack()

fenetre.mainloop()

Personnaliser un Button

bouton = tk.Button(
    fenetre,
    text="Clique ici",
    font=("Arial", 14),
    fg="white",
    bg="green",
    width=15,                # Largeur en caractères
    height=2                 # Hauteur en lignes
)
bouton.pack()

Entry (champ de saisie)

Un Entry permet à l'utilisateur de taper du texte.

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Entry exemple")

# Créer un champ de saisie
champ = tk.Entry(fenetre)
champ.pack()

fenetre.mainloop()

Récupérer le texte d'un Entry

texte_saisi = champ.get()  # Récupère le texte

Placer les widgets

Il existe plusieurs façons de placer les widgets dans une fenêtre :

1. pack() - Placement automatique

widget.pack()           # Centre par défaut
widget.pack(side="top")    # En haut
widget.pack(side="bottom") # En bas
widget.pack(side="left")   # À gauche
widget.pack(side="right")  # À droite

2. grid() - Placement en grille

widget.grid(row=0, column=0)  # Ligne 0, colonne 0
widget.grid(row=1, column=1)  # Ligne 1, colonne 1

3. place() - Placement précis

widget.place(x=50, y=100)  # Position exacte en pixels
💡 Conseil : Pour débuter, utilise pack(), c'est le plus simple !

Fonctions et événements

Pour qu'un bouton fasse quelque chose quand on clique dessus, il faut créer une fonction et la lier au bouton avec command.

Exemple : Afficher un message au clic

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Bouton interactif")

# Définir la fonction à exécuter
def dire_bonjour():
    print("Bonjour !")

# Créer le bouton et lier la fonction
bouton = tk.Button(fenetre, text="Clique-moi !", command=dire_bonjour)
bouton.pack()

fenetre.mainloop()
⚠️ Attention : Écris command=dire_bonjour sans parenthèses !
Si tu écris command=dire_bonjour(), la fonction s'exécute immédiatement au lieu d'attendre le clic.

Exemple : Modifier un Label au clic

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Modifier un texte")

# Créer le label
label = tk.Label(fenetre, text="Texte initial")
label.pack()

# Fonction qui modifie le label
def changer_texte():
    label.config(text="Nouveau texte !")

# Créer le bouton
bouton = tk.Button(fenetre, text="Changer", command=changer_texte)
bouton.pack()

fenetre.mainloop()
💡 label.config(text="...") permet de modifier le texte d'un Label après sa création.

Exemple : Utiliser un Entry avec un Button

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Salutation personnalisée")

# Créer les widgets
champ = tk.Entry(fenetre)
champ.pack()

label = tk.Label(fenetre, text="")
label.pack()

# Fonction qui utilise le texte saisi
def afficher_salutation():
    nom = champ.get()
    label.config(text=f"Bonjour {nom} !")

bouton = tk.Button(fenetre, text="Valider", command=afficher_salutation)
bouton.pack()

fenetre.mainloop()

Résumé des commandes importantes

Élément Code Description
Fenêtre tk.Tk() Créer la fenêtre principale
Label tk.Label(fenetre, text="...") Afficher du texte
Button tk.Button(fenetre, text="...", command=fonction) Créer un bouton cliquable
Entry tk.Entry(fenetre) Champ de saisie
Récupérer texte champ.get() Lire le contenu d'un Entry
Modifier Label label.config(text="...") Changer le texte d'un Label
Placer widget widget.pack() Afficher le widget
Boucle principale fenetre.mainloop() Garder la fenêtre ouverte

Exercice 1 - Première fenêtre

Consigne :

Crée une fenêtre avec :

  • Le titre : "Mon application"
  • Une taille de 300x200 pixels
  • Un Label affichant "Bienvenue dans mon programme !"
💡 N'oublie pas : import tkinter as tk, puis fenetre = tk.Tk(), et termine par fenetre.mainloop()

Exercice 2 - Bouton "Bonjour"

Consigne :

Crée un programme avec :

  • Un Label qui affiche "Rien pour le moment"
  • Un Button avec le texte "Dire bonjour"
  • Quand on clique sur le bouton, le Label change pour afficher "Bonjour !"
💡 Crée une fonction qui modifie le Label avec label.config(text="Bonjour !"), puis lie cette fonction au bouton avec command=ta_fonction

Exercice 3 - Messages différents ⭐

Consigne :

Crée un programme avec :

  • Un Label qui affiche un message
  • Trois boutons différents : "Bonjour", "Au revoir", "Comment ça va ?"
  • Chaque bouton affiche un message différent dans le Label quand on clique dessus

Exemple de résultat :

  • Clic sur "Bonjour" → Label affiche : "Salut ! 👋"
  • Clic sur "Au revoir" → Label affiche : "À bientôt ! 😊"
  • Clic sur "Comment ça va ?" → Label affiche : "Je vais bien, merci !"
💡 Crée trois fonctions différentes, une pour chaque bouton :
- dire_bonjour() qui change le Label en "Salut ! 👋"
- dire_au_revoir() qui change le Label en "À bientôt ! 😊"
- demander_nouvelles() qui change le Label en "Je vais bien, merci !"

Puis crée trois boutons, chacun avec sa propre fonction dans command.

Exercice 4 - Compteur de clics

Consigne :

Crée un programme avec :

  • Un Label qui affiche "Nombre de clics : 0"
  • Un Button avec le texte "Clique ici"
  • À chaque clic, le nombre augmente : "Nombre de clics : 1", "Nombre de clics : 2", etc.
💡 Crée une variable compteur = 0 avant la fenêtre. Dans la fonction, utilise global compteur pour la modifier, puis augmente-la de 1 et mets à jour le Label.

Exercice 5 - Saisie de nom

Consigne :

Crée un programme avec :

  • Un Label qui dit "Entre ton nom :"
  • Un Entry (champ de texte)
  • Un Button avec le texte "Valider"
  • Un Label vide en bas
  • Quand on clique sur "Valider", le Label du bas affiche : "Bonjour [nom] !"
💡 Utilise champ.get() pour récupérer le texte saisi, puis affiche-le dans le Label avec label.config(text=f"Bonjour {nom} !")

Exercice 6 - Calculatrice simple

Consigne :

Crée une calculatrice simple avec :

  • Deux Entry pour saisir deux nombres
  • Un Button "Additionner"
  • Un Label qui affiche le résultat : "Résultat : ..."

Quand on clique sur le bouton, le programme additionne les deux nombres et affiche le résultat.

💡 N'oublie pas de convertir les textes en nombres avec int() ou float() avant de les additionner :
nombre1 = float(champ1.get())

Exercice 7 - Jeu de devinette

Consigne :

Crée un petit jeu où l'utilisateur doit deviner un nombre entre 1 et 10 :

  • Le programme choisit un nombre secret (par exemple 7)
  • Un Label qui dit "Devine le nombre (entre 1 et 10)"
  • Un Entry pour que l'utilisateur tape sa proposition
  • Un Button "Vérifier"
  • Un Label qui affiche :
    • "Trop petit !" si le nombre est trop petit
    • "Trop grand !" si le nombre est trop grand
    • "Bravo ! Tu as trouvé !" si c'est le bon nombre
💡 Crée une variable nombre_secret = 7 au début. Dans la fonction, récupère la proposition avec int(champ.get()), puis compare-la avec nombre_secret en utilisant des if/elif/else.
