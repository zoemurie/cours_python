🎮 Créer un jeu Pong avec Tkinter

Ton premier jeu vidéo en Python !

Introduction au jeu Pong

Pong est l'un des premiers jeux vidéo de l'histoire (1972) ! C'est un jeu simple où deux joueurs contrôlent des raquettes pour renvoyer une balle, comme au tennis de table.

Dans ce cours, tu vas apprendre à créer ton propre jeu Pong en utilisant Python et Tkinter. C'est un excellent projet pour comprendre :

  • Comment créer des animations
  • Comment détecter les collisions
  • Comment gérer les contrôles au clavier
  • Comment créer une boucle de jeu

Principe du jeu

Règles du Pong :

  1. Une balle se déplace sur l'écran
  2. Deux raquettes (une à gauche, une à droite) peuvent monter et descendre
  3. Les joueurs doivent renvoyer la balle avec leur raquette
  4. Si la balle passe derrière une raquette, l'adversaire marque un point
  5. Le premier à atteindre un certain score gagne

Éléments nécessaires pour créer Pong

Les composants du jeu :

Élément Description Type sur Canvas
Terrain Le fond du jeu Canvas avec couleur de fond
Balle Cercle qui se déplace Ovale (circle)
Raquette gauche Rectangle contrôlé par le joueur 1 Rectangle
Raquette droite Rectangle contrôlé par le joueur 2 Rectangle
Scores Affichage des points Texte (Label ou canvas text)

Canvas pour le jeu

Créer le terrain de jeu

import tkinter as tk

fenetre = tk.Tk()
fenetre.title("Pong")

# Créer le Canvas (terrain de jeu)
canvas = tk.Canvas(fenetre, width=800, height=600, bg="black")
canvas.pack()

fenetre.mainloop()
💡 Pour un jeu, on utilise souvent un fond noir ou sombre, comme les jeux d'arcade classiques !

Créer les objets du jeu

# Créer la balle (cercle blanc)
balle = canvas.create_oval(390, 290, 410, 310, fill="white")

# Créer la raquette gauche (rectangle blanc)
raquette_gauche = canvas.create_rectangle(30, 250, 40, 350, fill="white")

# Créer la raquette droite
raquette_droite = canvas.create_rectangle(760, 250, 770, 350, fill="white")

Déplacer des objets sur le Canvas

La méthode move()

Pour déplacer un objet sur le Canvas, on utilise canvas.move(objet, dx, dy) :

  • objet : L'objet à déplacer (balle, raquette...)
  • dx : Déplacement horizontal (positif = droite, négatif = gauche)
  • dy : Déplacement vertical (positif = bas, négatif = haut)
# Déplacer la balle de 5 pixels vers la droite et 3 vers le bas
canvas.move(balle, 5, 3)

# Déplacer la raquette de 10 pixels vers le haut
canvas.move(raquette_gauche, 0, -10)

Récupérer la position d'un objet

Pour savoir où se trouve un objet, on utilise canvas.coords(objet) :

position = canvas.coords(balle)
# Retourne : [x1, y1, x2, y2] (coin haut-gauche et coin bas-droit)

Contrôles clavier

Détecter les touches du clavier

Pour contrôler les raquettes avec le clavier, on utilise bind() :

def deplacer_raquette_gauche_haut(event):
    canvas.move(raquette_gauche, 0, -20)

def deplacer_raquette_gauche_bas(event):
    canvas.move(raquette_gauche, 0, 20)

# Lier les touches aux fonctions
fenetre.bind("<z>", deplacer_raquette_gauche_haut)
fenetre.bind("<s>", deplacer_raquette_gauche_bas)

Touches courantes pour Pong :

Joueur Monter Descendre
Joueur 1 (gauche) "<z>" ou "<w>" "<s>"
Joueur 2 (droite) "<Up>" (flèche haut) "<Down>" (flèche bas)

Détection de collisions

Rebond sur les murs (haut et bas)

position_balle = canvas.coords(balle)
y_haut = position_balle[1]  # Coordonnée Y du haut de la balle
y_bas = position_balle[3]   # Coordonnée Y du bas de la balle

# Si la balle touche le haut ou le bas
if y_haut <= 0 or y_bas >= 600:
    vitesse_y = -vitesse_y  # Inverser la direction verticale

Collision avec une raquette

Pour détecter si la balle touche une raquette, on compare leurs positions :

pos_balle = canvas.coords(balle)
pos_raquette = canvas.coords(raquette_gauche)

# Vérifier si la balle touche la raquette gauche
if (pos_balle[0] <= pos_raquette[2] and  # La balle est à gauche de la raquette
    pos_balle[2] >= pos_raquette[0] and  # La balle est à droite du bord gauche
    pos_balle[3] >= pos_raquette[1] and  # La balle est sous le haut de la raquette
    pos_balle[1] <= pos_raquette[3]):    # La balle est au-dessus du bas
    vitesse_x = -vitesse_x  # Inverser la direction horizontale
💡 Cette vérification vérifie si les rectangles (balle et raquette) se chevauchent.

Boucle de jeu

Un jeu a besoin d'une boucle de jeu qui met à jour l'écran en continu (comme une animation). On utilise after() pour créer cette boucle : 

def boucle_jeu():
    # Déplacer la balle
    canvas.move(balle, vitesse_x, vitesse_y)
    
    # Vérifier les collisions
    # ... (code de collision)
    
    # Rappeler cette fonction après 20 millisecondes
    fenetre.after(20, boucle_jeu)

# Démarrer la boucle
boucle_jeu()
⚠️ fenetre.after(20, boucle_jeu) signifie : "Appelle la fonction boucle_jeu dans 20 millisecondes". Cela crée une boucle infinie qui rafraîchit le jeu environ 50 fois par seconde.

Structure complète d'un jeu Pong

import tkinter as tk

# Créer la fenêtre
fenetre = tk.Tk()
fenetre.title("Pong")

# Créer le Canvas
canvas = tk.Canvas(fenetre, width=800, height=600, bg="black")
canvas.pack()

# Créer les objets
balle = canvas.create_oval(390, 290, 410, 310, fill="white")
raquette_gauche = canvas.create_rectangle(30, 250, 40, 350, fill="white")
raquette_droite = canvas.create_rectangle(760, 250, 770, 350, fill="white")

# Variables de vitesse
vitesse_x = 5
vitesse_y = 3

# Fonctions de contrôle des raquettes
def raquette_gauche_haut(event):
    canvas.move(raquette_gauche, 0, -20)

def raquette_gauche_bas(event):
    canvas.move(raquette_gauche, 0, 20)

def raquette_droite_haut(event):
    canvas.move(raquette_droite, 0, -20)

def raquette_droite_bas(event):
    canvas.move(raquette_droite, 0, 20)

# Lier les touches
fenetre.bind("<z>", raquette_gauche_haut)
fenetre.bind("<s>", raquette_gauche_bas)
fenetre.bind("<Up>", raquette_droite_haut)
fenetre.bind("<Down>", raquette_droite_bas)

# Boucle de jeu
def boucle_jeu():
    global vitesse_x, vitesse_y
    
    # Déplacer la balle
    canvas.move(balle, vitesse_x, vitesse_y)
    
    # Récupérer la position de la balle
    pos = canvas.coords(balle)
    
    # Rebond haut/bas
    if pos[1] <= 0 or pos[3] >= 600:
        vitesse_y = -vitesse_y
    
    # Rebond gauche/droite (simplifié)
    if pos[0] <= 0 or pos[2] >= 800:
        vitesse_x = -vitesse_x
    
    # Continuer la boucle
    fenetre.after(20, boucle_jeu)

# Démarrer le jeu
boucle_jeu()

fenetre.mainloop()
💡 Ceci est une version simplifiée. Dans les exercices, tu vas construire un jeu complet avec collisions de raquettes et système de score !

Exercice 3 - Déplacer la balle

Consigne :

Fais bouger la balle automatiquement :

  • Crée deux variables vitesse_x = 5 et vitesse_y = 3
  • Crée une fonction deplacer_balle() qui :
    • Déplace la balle avec canvas.move()
    • Se rappelle elle-même après 20 millisecondes avec fenetre.after()
  • Appelle la fonction pour démarrer l'animation
💡 Structure de la fonction :
def deplacer_balle():
    canvas.move(balle, vitesse_x, vitesse_y)
    fenetre.after(20, deplacer_balle)

Exercice 4 - Rebonds sur les murs

Consigne :

Fais rebondir la balle sur les murs haut et bas :

  • Dans la fonction deplacer_balle(), récupère la position de la balle avec canvas.coords(balle)
  • Si la balle touche le haut (y ≤ 0) ou le bas (y ≥ 600), inverse vitesse_y
  • Si la balle touche la gauche (x ≤ 0) ou la droite (x ≥ 800), inverse vitesse_x
💡 Utilise global vitesse_x, vitesse_y au début de la fonction pour modifier ces variables
💡 Pour inverser : vitesse_y = -vitesse_y

Exercice 5 - Créer les raquettes

Consigne :

Ajoute les deux raquettes :

  • Raquette gauche : Rectangle blanc de 10×100 pixels, positionné à x=30, y=250
  • Raquette droite : Rectangle blanc de 10×100 pixels, positionné à x=760, y=250
💡 raquette_gauche = canvas.create_rectangle(30, 250, 40, 350, fill="white")
💡 raquette_droite = canvas.create_rectangle(760, 250, 770, 350, fill="white")

Exercice 6 - Contrôler les raquettes

Consigne :

Permets aux joueurs de contrôler les raquettes avec le clavier :

  • Joueur 1 (gauche) : Touche Z pour monter, S pour descendre
  • Joueur 2 (droite) : Flèche Haut pour monter, flèche Bas pour descendre
  • Les raquettes se déplacent de 20 pixels à chaque pression
  • Les raquettes ne doivent pas sortir du Canvas (bloquer à y=0 et y=500)
💡 Crée 4 fonctions : raquette_gauche_haut(event), raquette_gauche_bas(event), etc.
💡 Utilise canvas.move(raquette, 0, -20) pour monter
💡 Vérifie la position avec canvas.coords() avant de déplacer pour ne pas sortir

Exercice 7 - Collision balle-raquette

Consigne :

Détecte quand la balle touche une raquette et fais-la rebondir :

  • Dans la fonction deplacer_balle(), récupère les positions de la balle et des deux raquettes
  • Vérifie si la balle chevauche la raquette gauche
  • Vérifie si la balle chevauche la raquette droite
  • Si collision détectée, inverse vitesse_x
💡 Condition de collision (exemple pour raquette gauche) :
if (pos_balle[0] <= pos_raquette[2] and
    pos_balle[2] >= pos_raquette[0] and
    pos_balle[3] >= pos_raquette[1] and
    pos_balle[1] <= pos_raquette[3]):
    vitesse_x = -vitesse_x

Exercice 8 - Système de score

Consigne :

Ajoute un système de score :

  • Crée deux variables score_gauche = 0 et score_droite = 0
  • Affiche les scores en haut du Canvas avec canvas.create_text()
  • Quand la balle sort à gauche (x ≤ 0), le joueur de droite marque un point
  • Quand la balle sort à droite (x ≥ 800), le joueur de gauche marque un point
  • Après un point, replace la balle au centre
  • Mets à jour l'affichage du score
💡 Crée les textes de score au début :
texte_score_gauche = canvas.create_text(200, 50, text="0", fill="white", font=("Arial", 40))
💡 Pour mettre à jour : canvas.itemconfig(texte_score_gauche, text=str(score_gauche))
💡 Pour recentrer la balle : canvas.coords(balle, 390, 290, 410, 310)

Structure du code :

Organisation recommandée :

1️⃣ Imports et configuration
   - import tkinter
   - créer fenêtre et canvas

2️⃣ Créer les objets
   - balle, raquettes, scores, ligne centrale

3️⃣ Variables globales
   - vitesse_x, vitesse_y
   - score_gauche, score_droite

4️⃣ Fonctions de contrôle des raquettes
   - raquette_gauche_haut(), raquette_gauche_bas()
   - raquette_droite_haut(), raquette_droite_bas()

5️⃣ Fonction boucle de jeu
   - déplacer la balle
   - vérifier rebonds sur murs
   - vérifier collisions avec raquettes
   - vérifier si un point est marqué
   - rappeler la fonction avec after()

6️⃣ Lier les touches et démarrer
   - bind() pour les 4 touches
   - appeler boucle_jeu()
   - mainloop()

Points importants :

⚠️ Erreurs fréquentes à éviter :
  • Oublier global vitesse_x, vitesse_y dans la boucle de jeu
  • Ne pas vérifier si les raquettes sortent du terrain
  • Oublier de remettre la balle au centre après un point
  • Ne pas mettre à jour l'affichage du score
  • Utiliser after() avec un délai trop grand (le jeu sera lent)
