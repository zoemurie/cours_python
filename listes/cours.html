Cours – Les listes en Python

Qu’est-ce qu’une liste ?

Une liste est une variable qui peut contenir plusieurs valeurs. Au lieu d’avoir plusieurs variables : 

note1 = 12
note2 = 15
note3 = 9

On peut utiliser une seule liste :

notes = [12, 15, 9]
Une seule variable, plusieurs valeurs.

Créer une liste

nombres = [1, 2, 3, 4]
mots = ["chat", "chien", "oiseau"]
vide = []

Une liste peut contenir :

  • des nombres
  • des mots
  • ou être vide

Les positions dans une liste (indices)

Important : les indices commencent à 0 
fruits = ["pomme", "banane", "orange"]

Position et valeur :

  • 0 : pomme
  • 1 : banane
  • 2 : orange

Exemples :

print(fruits[0])  # affiche pomme
print(fruits[2])  # affiche orange

Ajouter un élément dans une liste

fruits.append("fraise")

L’élément est ajouté à la fin de la liste.

Parcourir une liste avec une boucle for

fruits = ["pomme", "banane", "orange"]

for fruit in fruits:
    print(fruit)

La boucle affiche chaque élément de la liste.

Connaître la taille d’une liste

print(len(fruits))

La fonction len() renvoie le nombre d’éléments dans la liste.

À retenir

  • Une liste stocke plusieurs valeurs
  • Les indices commencent à 0
  • append() permet d’ajouter un élément
  • Une boucle for permet de parcourir la liste
