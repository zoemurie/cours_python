Le module Turtle en Python

1. À quoi sert Turtle ?

Le module Turtle permet de dessiner avec Python. On contrôle une petite tortue qui se déplace à l’écran et trace des lignes.

Turtle est très utile pour :

  • apprendre la programmation de manière visuelle
  • comprendre les boucles et les angles
  • dessiner des formes géométriques

2. Importer Turtle

Avant d’utiliser Turtle, il faut importer le module. 

import turtle

3. Créer la tortue

On crée ensuite une tortue que l’on va contrôler. 

t = turtle.Turtle()

4. Avancer et tourner

La tortue peut avancer et tourner.

  • forward(distance) : avance
  • right(angle) : tourne à droite
  • left(angle) : tourne à gauche
t.forward(100)
t.right(90)
t.forward(100)

5. Dessiner un carré

En combinant déplacements et rotations, on peut dessiner des formes. 

import turtle

t = turtle.Turtle()

t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)

6. Utiliser une boucle

Pour éviter de répéter le même code, on utilise une boucle for. 

import turtle

t = turtle.Turtle()

for i in range(4):
    t.forward(100)
    t.right(90)

7. Changer la vitesse et la couleur

  • speed() : change la vitesse
  • color() : change la couleur
t.speed(3)
t.color("blue")

8. Lever et baisser le stylo

La tortue peut se déplacer sans dessiner.

  • penup() : lève le stylo
  • pendown() : baisse le stylo
t.penup()
t.forward(50)
t.pendown()
t.forward(100)

9. Garder la fenêtre ouverte

Pour éviter que la fenêtre se ferme automatiquement : 

turtle.done()

10. À retenir absolument

  • Turtle permet de dessiner avec Python
  • On contrôle une tortue avec des commandes simples
  • Les boucles permettent de dessiner plus facilement
  • C’est un excellent outil pour débuter
