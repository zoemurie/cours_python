Comprendre les fonctions en Python

I – Qu’est-ce qu’une fonction ?

Une fonction est un groupe d’instructions auquel on donne un nom pour pouvoir les utiliser plusieurs fois. Au lieu de répéter du code, on le place dans une fonction.

Exemple sans fonction

print("Bonjour")
print("Bonjour")
print("Bonjour")

Exemple avec une fonction

def dire_bonjour():
    print("Bonjour")

dire_bonjour()
dire_bonjour()
dire_bonjour()
On définit une fonction avec def et on l’appelle par son nom.

Règles importantes

  • Une fonction se définit avec def : 
    def ma_fonction():
    ...
  • Le code à l’intérieur de la fonction doit être indenté : 
    def test():
    print("Dans la fonction")
  • Une fonction ne s’exécute que si on l’appelle : 
    test()

Pourquoi les fonctions sont utiles dans un jeu ?

Dans un jeu, plusieurs actions peuvent se répéter :

  • cliquer sur un bouton
  • déplacer un personnage
  • augmenter un score

Chaque action peut être définie comme une fonction.

II – Les paramètres

1. Fonction simple (sans paramètre)

def dire_bonjour():
    print("Bonjour")

Pour l’utiliser :

dire_bonjour()

2. Fonctions avec paramètres

Un paramètre est une information que l’on donne à une fonction pour qu’elle puisse travailler avec. 

def dire_bonjour(prenom):
    print("Bonjour", prenom)

Exemples d’appels :

dire_bonjour("Alice")
dire_bonjour("Lucas")

3. Plusieurs paramètres

def addition(a, b):
    print(a + b)

4. print ou return ?

print : affiche un résultat, mais ne renvoie rien. 
def addition(a, b):
    print(a + b)
return : renvoie un résultat, ce qui permet de le réutiliser plus tard. 
def addition(a, b):
    return a + b
Exemple d’utilisation : 
resultat = addition(3, 4)
print(resultat)
