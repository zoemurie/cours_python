Exercices – Les fonctions en Python

Exercice 1 – Paramètres simples

Créer une fonction nommée dire_bonjour(prenom, nom). Cette fonction doit afficher un message de la forme : 

Bonjour prenom nom

Exemple d’appel :

dire_bonjour("Aya", "Nakamura")

Résultat attendu :

Bonjour Aya Nakamura

Exercice 2 – Calcul avec return

Créer une fonction division(nombre1, nombre2) qui retourne le résultat de la division de nombre1 par nombre2.

Exercice 3 – Comparaison

Créer une fonction est_majeur(age) qui :

  • retourne True si l’âge est supérieur ou égal à 18 ;
  • retourne False dans le cas contraire.

Exercice 4 – Valeur maximale

Créer une fonction maximum(a, b) qui retourne le plus grand des deux nombres.

Exercice 5 – Utilisation d’une fonction

Sans modifier la fonction précédente, utiliser son résultat pour afficher le message suivant : 

Le plus grand nombre est ...

Mini-projet – Préparer le jeu

Créer une fonction augmenter_score(score)

  • ajoute 1 point au score ;
  • retourne la nouvelle valeur du score ;
  • affiche le message Gagner ! si le score atteint 10.

Créer une fonction Fonction perdre_vie(vies)

  • retire 1 point de vie ;
  • retourne le nombre de vies restantes ;
  • affiche le message Perdu ! s’il ne reste plus de vie.
