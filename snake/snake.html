🐍 Créer un jeu Snake avec Tkinter

Le jeu classique du serpent gourmand

Introduction au jeu Snake

Snake (le serpent) est un jeu classique où tu contrôles un serpent qui doit manger de la nourriture pour grandir, tout en évitant de se mordre la queue ou de percuter les murs !

Ce jeu est parfait pour apprendre :

  • La gestion d'une grille
  • Les listes pour représenter le corps du serpent
  • Les déplacements dans 4 directions
  • La détection de collision
  • Le système de score

Principe du jeu

Règles du Snake :

  1. Un serpent se déplace sur une grille
  2. Le serpent avance automatiquement dans une direction
  3. Le joueur peut changer la direction avec les flèches du clavier
  4. Des fruits apparaissent aléatoirement sur la grille
  5. Quand le serpent mange un fruit :
    • Il grandit d'une case
    • Le score augmente
    • Un nouveau fruit apparaît
  6. Le jeu se termine si :
    • Le serpent se mord la queue
    • Le serpent touche un mur

Système de grille

Dans Snake, on ne travaille pas en pixels, mais avec un système de grille. Le terrain est divisé en cases (par exemple 20×20 cases).

Principe de la grille

# Paramètres de la grille
TAILLE_CASE = 20      # Taille d'une case en pixels
LARGEUR_GRILLE = 20   # Nombre de cases en largeur
HAUTEUR_GRILLE = 20   # Nombre de cases en hauteur

# Taille du Canvas
LARGEUR = LARGEUR_GRILLE * TAILLE_CASE   # 400 pixels
HAUTEUR = HAUTEUR_GRILLE * TAILLE_CASE   # 400 pixels

Dessiner une case sur la grille

def dessiner_case(x, y, couleur):
    """Dessine un rectangle à la position (x, y) de la grille"""
    x1 = x * TAILLE_CASE
    y1 = y * TAILLE_CASE
    x2 = x1 + TAILLE_CASE
    y2 = y1 + TAILLE_CASE
    canvas.create_rectangle(x1, y1, x2, y2, fill=couleur, outline="black")
💡 Si le serpent est en position (5, 3) sur la grille, il sera dessiné aux pixels (100, 60) sur le Canvas.

Représenter le serpent

Le serpent est représenté par une liste de positions. Chaque position est un tuple (x, y) qui correspond à une case de la grille.

Exemple : serpent de 3 cases

# Le serpent occupe 3 cases : (5,10), (6,10), (7,10)
serpent = [(5, 10), (6, 10), (7, 10)]

# La tête du serpent est toujours à la fin de la liste
tete = serpent[-1]  # (7, 10)
⚠️ Convention importante : La tête du serpent est toujours le dernier élément de la liste.

Dessiner le serpent

def dessiner_serpent():
    """Dessine toutes les parties du serpent"""
    for x, y in serpent:
        dessiner_case(x, y, "green")

Déplacement du serpent

Principe du mouvement

Pour déplacer le serpent :

  1. Calculer la position de la nouvelle tête selon la direction
  2. Ajouter cette nouvelle tête à la fin de la liste
  3. Retirer la queue (premier élément) pour que le serpent garde la même taille
# Direction actuelle
direction = "droite"

def deplacer_serpent():
    # Récupérer la tête actuelle
    tete_x, tete_y = serpent[-1]
    
    # Calculer la nouvelle position selon la direction
    if direction == "droite":
        nouvelle_tete = (tete_x + 1, tete_y)
    elif direction == "gauche":
        nouvelle_tete = (tete_x - 1, tete_y)
    elif direction == "haut":
        nouvelle_tete = (tete_x, tete_y - 1)
    elif direction == "bas":
        nouvelle_tete = (tete_x, tete_y + 1)
    
    # Ajouter la nouvelle tête
    serpent.append(nouvelle_tete)
    
    # Retirer la queue
    serpent.pop(0)

Changer de direction avec le clavier

def changer_direction(event):
    global direction
    touche = event.keysym
    
    # Empêcher de faire demi-tour
    if touche == "Up" and direction != "bas":
        direction = "haut"
    elif touche == "Down" and direction != "haut":
        direction = "bas"
    elif touche == "Left" and direction != "droite":
        direction = "gauche"
    elif touche == "Right" and direction != "gauche":
        direction = "droite"

# Lier les touches
fenetre.bind("<Key>", changer_direction)
⚠️ Important : Il faut empêcher le serpent de faire demi-tour (si il va à droite, il ne peut pas aller directement à gauche).

Ajouter la nourriture

Positionner un fruit aléatoire

import random

def placer_nourriture():
    """Place un fruit à une position aléatoire"""
    x = random.randint(0, LARGEUR_GRILLE - 1)
    y = random.randint(0, HAUTEUR_GRILLE - 1)
    return (x, y)

# Position de la nourriture
nourriture = placer_nourriture()

Dessiner la nourriture

def dessiner_nourriture():
    x, y = nourriture
    dessiner_case(x, y, "red")

Détection de collision

Vérifier si le serpent mange la nourriture

def verifier_nourriture():
    global nourriture, score
    tete = serpent[-1]
    
    if tete == nourriture:
        # Le serpent mange !
        score += 1
        nourriture = placer_nourriture()
        return True
    return False

Vérifier si le serpent se mord

def verifier_collision_soi():
    tete = serpent[-1]
    # Vérifier si la tête touche le corps (tous les éléments sauf la tête)
    return tete in serpent[:-1]

Vérifier si le serpent touche un mur

def verifier_collision_mur():
    tete_x, tete_y = serpent[-1]
    
    if tete_x < 0 or tete_x >= LARGEUR_GRILLE:
        return True
    if tete_y < 0 or tete_y >= HAUTEUR_GRILLE:
        return True
    return False

Faire grandir le serpent

Quand le serpent mange, il doit grandir. Pour cela, il suffit de ne pas retirer la queue lors du prochain déplacement. 

def deplacer_serpent():
    # ... (calculer nouvelle_tete)
    
    # Ajouter la nouvelle tête
    serpent.append(nouvelle_tete)
    
    # Vérifier si le serpent mange
    if not verifier_nourriture():
        # Si pas de nourriture mangée, retirer la queue
        serpent.pop(0)
    # Sinon, on garde la queue → le serpent grandit !

Game Over et boucle de jeu

Structure de la boucle principale

def boucle_jeu():
    global en_cours
    
    if not en_cours:
        return
    
    # Déplacer le serpent
    deplacer_serpent()
    
    # Vérifier les collisions
    if verifier_collision_mur() or verifier_collision_soi():
        en_cours = False
        canvas.create_text(200, 200, text="GAME OVER!", 
                          fill="red", font=("Arial", 40))
        return
    
    # Effacer le Canvas
    canvas.delete("all")
    
    # Redessiner tout
    dessiner_serpent()
    dessiner_nourriture()
    
    # Rappeler la fonction après 150ms
    fenetre.after(150, boucle_jeu)
💡 Plus la valeur dans after() est petite, plus le jeu est rapide. 150ms = vitesse moyenne, 100ms = rapide, 200ms = lent.
