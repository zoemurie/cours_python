Les conditions en Python
if / elif / else

1. Comparer des valeurs

Avant d’utiliser des conditions, il faut savoir comparer des valeurs. Une comparaison donne toujours un résultat : True (vrai) ou False (faux).

2. Les opérateurs de comparaison

  • == : égal à
  • != : différent de
  • > : plus grand que
  • < : plus petit que
  • >= : plus grand ou égal à
  • <= : plus petit ou égal à
print(5 > 3)      # True
print(10 == 8)   # False
print(4 <= 4)    # True

3. À quoi sert une condition ?

Une condition permet à un programme de prendre une décision. Selon le résultat de la comparaison, le programme n’exécute pas le même code.

Exemple simple dans la vie courante :

  • Si le feu est vert → je passe
  • Sinon → je m’arrête

4. La condition if

if signifie « si ». Le code à l’intérieur s’exécute seulement si la condition est vraie. 

age = 20

if age >= 18:
    print("Majeur")
À respecter absolument :
  • La condition se termine par :
  • Le code à l’intérieur est indenté

5. La condition else

else signifie « sinon ». Il s’exécute si la condition du if est fausse. 

age = 15

if age >= 18:
    print("Majeur")
else:
    print("Mineur")

6. La condition elif

elif signifie « sinon si ». On l’utilise lorsqu’il y a plusieurs cas possibles. 

note = 12

if note >= 16:
    print("Très bien")
elif note >= 10:
    print("Moyen")
else:
    print("Insuffisant")
À retenir :
  • Un seul if
  • Autant de elif que nécessaire
  • Un seul else
  • Une seule partie est exécutée

7. Exemple très simple avec un mot

mot = "python"

if mot == "python":
    print("Mot correct")
else:
    print("Mot incorrect")

8. À retenir absolument

  • Les comparaisons donnent True ou False
  • if teste une condition
  • elif ajoute des cas
  • else est le dernier cas
  • L’indentation est obligatoire
