Les Structures Conditionnelles en Python

Introduction

Les structures conditionnelles permettent à un programme de prendre des décisions et d'exécuter différentes instructions selon les situations. C'est comme un carrefour où le programme choisit quelle route prendre.

1. Les Opérateurs de Comparaison

Avant d'utiliser les conditions, il faut savoir comparer des valeurs. Voici les opérateurs disponibles :

Opérateur Signification Exemple
== Égal à 5 == 5
!= Différent de 5 != 3
< Inférieur à 3 < 5
> Supérieur à 7 > 5
<= Inférieur ou égal à 5 <= 5
>= Supérieur ou égal à 8 >= 5

2. La Structure if (Si)

La structure if permet d'exécuter un bloc de code uniquement si une condition est vraie.

Syntaxe :

if condition:
    # Instructions à exécuter si la condition est vraie

Exemple :

age = 14
if age >= 13:
    print("Tu es un adolescent !")
⚠️ IMPORTANT : L'indentation (les espaces au début de la ligne) est OBLIGATOIRE en Python ! Utilise toujours 4 espaces pour indenter le code à l'intérieur du if.

3. La Structure else (Sinon)

La structure else permet d'exécuter un bloc de code si la condition du if est fausse.

Syntaxe :

if condition:
    # Instructions si vrai
else:
    # Instructions si faux

Exemple :

temperature = 18
if temperature > 20:
    print("Il fait bon !")
else:
    print("Il fait frais.")

4. La Structure elif (Sinon si)

La structure elif (contraction de "else if") permet de tester plusieurs conditions différentes. On peut utiliser autant de elif que nécessaire.

Syntaxe :

if condition1:
    # Instructions si condition1 est vraie
elif condition2:
    # Instructions si condition2 est vraie
else:
    # Instructions si aucune condition n'est vraie

Exemple :

note = 15
if note >= 16:
    print("Très bien !")
elif note >= 14:
    print("Bien !")
elif note >= 10:
    print("Assez bien.")
else:
    print("Tu dois travailler davantage.")

5. Comment ça fonctionne ?

Python teste les conditions dans l'ordre :

  1. Il teste d'abord la condition du if
  2. Si elle est vraie, il exécute le code du if et saute le reste
  3. Si elle est fausse, il passe au premier elif (s'il existe)
  4. Il continue ainsi jusqu'à trouver une condition vraie
  5. Si aucune condition n'est vraie, il exécute le else (s'il existe)

Points à retenir :

  • N'oublie pas les deux-points (:) après chaque condition
  • L'indentation est obligatoire (4 espaces)
  • Une seule branche est exécutée (la première condition vraie)
  • Le else est facultatif
  • On peut avoir plusieurs elif mais un seul if et un seul else
