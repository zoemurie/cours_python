Exercice 2 – Pair ou impair

Écris un programme pair_ou_impair.py qui :

  • crée une variable nombre
  • affiche "pair" si le nombre est divisible par 2
  • sinon affiche "impair"

Indice : utiliser % 2

L’opérateur %

En Python (et dans beaucoup d’autres langages), % s’appelle l’opérateur modulo. Il sert à calculer le reste d’une division entière (division euclidienne).

Exemples :

7 % 2 = 1   # 7 = 2 x 3 + 1 → reste 1
10 % 2 = 0  # 10 = 2 x 5 + 0 → reste 0
15 % 3 = 0  # 15 = 3 x 5 + 0 → reste 0
14 % 5 = 4  # 14 = 5 x 2 + 4 → reste 4

Donc :

  • Si nombre % 2 == 0, le nombre est divisible par 2 → pair
  • Sinon, il reste 1 → impair
