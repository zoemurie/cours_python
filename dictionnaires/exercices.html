Fiche d’exercices – Les dictionnaires en Python

Exercice 1 – Créer un dictionnaire

Créer un dictionnaire animal contenant :

  • "nom" : "chat"
  • "age" : 3

Puis afficher le dictionnaire.

Exercice 2 – Lire une valeur

Avec le dictionnaire :

personne = {"prenom": "Lucas", "age": 14}

Afficher :

  • le prénom
  • l’âge

Exercice 3 – Ajouter des informations

Créer un dictionnaire vide eleve et ajouter :

  • le prénom
  • la classe
  • la note

Puis afficher le dictionnaire.

Exercice 4 – Modifier une valeur

Créer :

score = {"joueur": "Alex", "points": 10}

Modifier les points pour qu’ils valent 15.

Exercice 5 – Parcourir un dictionnaire

Créer un dictionnaire de matières et de notes :

notes = {"maths": 14, "francais": 12, "anglais": 16}

Utiliser une boucle for pour afficher :

maths : 14
francais : 12
anglais : 16

Exercice 6 – Compter les éléments

Créer un dictionnaire avec plusieurs informations et afficher le nombre d’éléments qu’il contient.

Exercice 7 – Conditions et dictionnaires

Créer un dictionnaire eleve avec une note.

Si la note est :

  • ≥ 10 → afficher "Réussi"
  • Sinon → afficher "Échec"

Exercice 8 – Mini défi

Créer un dictionnaire contact contenant :

  • nom
  • telephone
  • email

Afficher toutes les informations avec une boucle.

Exercice 9 – Dictionnaire + saisie utilisateur

Créer un dictionnaire vide. Demander à l’utilisateur :

  • son prénom
  • son âge

Stocker ces informations dans le dictionnaire puis l’afficher.

Projet de fin – Répertoire d’élèves

Objectif : Créer un programme permettant de gérer un répertoire d’élèves avec des dictionnaires, un menu interactif, des boucles et des conditions.

Structure attendue

Dictionnaire principal eleves où chaque élève est identifié par son prénom, et ses informations sont stockées dans un dictionnaire :

eleves = {
    "Alice": {"age": 12, "classe": "5A", "moyenne": 14},
    "Lucas": {"age": 13, "classe": "5B", "moyenne": 11}
}

Consignes générales

  • Utiliser des dictionnaires imbriqués
  • Utiliser while, for, if / elif / else
  • Le programme fonctionne tant que l’utilisateur ne choisit pas de quitter

Étapes du projet

1 – Menu principal

1 - Ajouter un élève
2 - Afficher tous les élèves
3 - Afficher les informations d'un élève
4 - Modifier la moyenne d'un élève
5 - Supprimer un élève
6 - Quitter

2 – Ajouter un élève

  • Demander le prénom
  • Demander l’âge
  • Demander la classe
  • Demander la moyenne
  • Ajouter l’élève dans le dictionnaire

3 – Afficher tous les élèves

Afficher la liste des prénoms. Si aucun élève n’est enregistré, afficher "Aucun élève".

4 – Informations d’un élève

Demander un prénom. Si l’élève existe, afficher toutes ses informations. Sinon, afficher "Élève introuvable".

5 – Modifier la moyenne

Demander un prénom. Si l’élève existe, demander la nouvelle moyenne et modifier la valeur. Sinon, afficher "Élève introuvable".

6 – Supprimer un élève

Demander un prénom. Supprimer l’élève s’il existe. Sinon, afficher "Élève introuvable".

7 – Quitter

Afficher "Au revoir" et arrêter le programme.

Bonus (facultatif)

  • Afficher la moyenne générale de la classe
  • Afficher les élèves avec une moyenne ≥ 10
  • Compter le nombre d’élèves
  • Ajouter une option sauvegarde / chargement

Conseils pour réussir

  • Commencer par faire fonctionner le menu
  • Ajouter les options une par une
  • Tester souvent
  • Bien respecter l’indentation
