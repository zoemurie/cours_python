Cours – Les dictionnaires en Python

Qu’est-ce qu’un dictionnaire ?

Un dictionnaire est une variable qui permet d’associer une information à une autre. Les données sont stockées sous la forme clé → valeur.

Créer un dictionnaire

eleve = {"prenom": "Alice", "age": 12, "note": 15}
Ici :
  • eleve est le dictionnaire
  • "prenom", "age", "note" sont les clés
  • "Alice", 12, 15 sont les valeurs

Accéder à une valeur

On utilise la clé :

print(eleve["prenom"])  # affiche Alice
print(eleve["age"])     # affiche 12
La clé doit exister, sinon le programme plante.

Ajouter ou modifier une information

Pour ajouter une information :

eleve["classe"] = "5A"

Pour modifier une valeur :

eleve["age"] = 13

Supprimer une information

del eleve["note"]

Parcourir un dictionnaire

Parcourir uniquement les clés :

for cle in eleve:
    print(cle)

Parcourir clés et valeurs :

for cle, valeur in eleve.items():
    print(cle, ":", valeur)

Taille d’un dictionnaire

print(len(eleve))

Affiche le nombre d’informations stockées dans le dictionnaire.

À retenir

  • Un dictionnaire associe une clé à une valeur.
  • On accède aux données avec les clés.
  • Les clés sont souvent des mots.
  • Les dictionnaires sont très utiles pour organiser des informations.
